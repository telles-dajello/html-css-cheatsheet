Table tags

Tag Name Description
<table> Table The wrapper for the table as a whole
<thead> Table Head The tag that defines the first row with titles for each column
<tbody> Table Body tag defining the area of rows between head and a possible footer
<tfoot> Table Foot The tag defining the foot row
<tr> Table Row Tag that defines the set of cells (columns in the row) for a row of the table
<td> Table Data Tag that defines the content of each cell
<th> Table Heading Similar to Table Data, but used in the Table Head

CSS selectors and properties

CSS selectors

Name Examples Description
ID #my-id IDs are the most specific selector and have the highest precedence. They are denoted by a "#" symbol followed by the ID name
Class .my-class A class selector selects elements based on the value of their class attribute class="my-class"
Type p or h1 or span A type selector selects elements based on their HTML tag name.
Universal * The universal selector selects all elements on the page, regardless of their tag name, class, or ID.
Pseudo-Class :hover A pseudo-class selector selects elements based on their state or position in the document.

CSS properties

Property Description
<color> This property sets the color of text inside an element. The value can be specified in a number of ways, including named colors (e.g. "red"), RGB values, and hexadecimal values.
<font-size> This property sets the size of text inside an element. The value can be specified in pixels, ems, rems, or other units.
<background-color> This property sets the background color of an element. The value can be specified in the same ways as the color property.
<margin> This property sets the space outside an element, between its border and the adjacent elements. The value can be specified as a single value (to set all margins equally), or as multiple values (to set margins for each side separately).
<padding> This property sets the space inside an element, between its content and its border. The value can be specified in the same way as the margin property.
<border> This property sets the border of an element. It can be used to set the width, style, and color of the border. The value can be specified as a shorthand (to set all three properties at once) or as separate properties.
<display> This property sets how an element should be displayed on the page. It can be used to make elements block-level or inline, or to hide them from the page altogether.
<width> This property sets the width of an element. The value can be specified in pixels, percentages, or other units.
<height> This property sets the height of an element. The value can be specified in the same way as the width property.
<text-align> This property sets the horizontal alignment of text within an element. The value can be specified as left, right, center, or justified.
